# Vouch Digital Task NodeJs

## To Run this project

### Locally

- Clone this repository.
- Create a .env file in the root of the project.
- Enter MONGO_URI and JWT_KEY secrets in the .env file.
- Run => npm install
- Run => npm run build
- Run => npm run start
